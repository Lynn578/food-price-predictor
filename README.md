# Food Price Predictor - Kenya

A Streamlit web application that helps users analyze food prices, view trends, predict future prices, and get buying recommendations for grocery items in Kenya.

## Features

- **📊 See Food Prices**: View current prices across different markets
- **📈 Price Trends**: Visualize price changes over time with interactive charts
- **🔮 Predict Future Price**: Forecast future prices using historical patterns
- **💡 Buying Suggestions**: Get smart recommendations based on price trends and your budget
